# @flupy/core

Protocol-level primitives shared by <code>@flupy/browser</code> and <code>@flupy/react</code>. Pure TypeScript — no browser APIs, no React, no wallet dependency.

<h2 id="encoding">Proof Encoding</h2>

SnarkJS outputs Groth16 proof points as decimal strings. Soroban contracts expect fixed-size byte arrays (<code>BytesN&lt;64&gt;</code>, <code>BytesN&lt;128&gt;</code>, <code>BytesN&lt;32&gt;</code>). The encoding helpers in <code>@flupy/core</code> handle this conversion:
